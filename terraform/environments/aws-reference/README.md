# ⚠️ AWS 프로덕션 레퍼런스 — 참고용 코드

> **WARNING**: 이 디렉토리의 Terraform 코드는 **실행 시 실제 AWS 비용이 발생**합니다.

## 용도

학습 및 아키텍처 설계 참고 목적입니다. 실제 클라우드 인프라를 프로비저닝하고 싶다면:

1. AWS 계정 생성
2. AWS CLI 설정 (`aws configure`)
3. `terraform.tfvars`에 민감 정보 설정
4. `terraform plan`으로 예상 비용 확인 후 `terraform apply`

## 예상 비용 (서울 리전, 최소 사양)

| 리소스 | 사양 | 월 비용 |
|--------|------|---------|
| EC2 | t3.micro | ~$10 |
| RDS | db.t3.micro | ~$15 |
| ALB | 기본 | ~$20 |
| **합계** | | **~$45/월** |
